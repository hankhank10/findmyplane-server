Features: Simple, free flight tracking

Easy to Use

No complex setup, no configuration - just download, run the program and type in the short code on any browser

Access on any device

View your map on Chrome, Firefox or Safari on Windows, Mac, iPad, Android - anywhere that runs a browser

Access on any network

Connects through the internet, so view your map even when your secondary device is not connected to the same network

Free and open source

No charge to download or use.

No logins or passwords

No need to create an account or remember a password. Every time you start a flight you are given a unique five digit ident code. Type it in on this site and you have your map.
